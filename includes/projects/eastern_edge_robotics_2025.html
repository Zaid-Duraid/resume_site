Eastern Edge Robotics 2025

Eastern Edge Robotics' 2025 remotely-operated vehicle (ROV), built for the MATE competition, is named Waterwitch. See this project's repository

Waterwitch features a Raspberry Pi 4 in the enclosure along with an RP2040 microcontroller and electronic speed controllers. As the software co-lead, I contributed to all aspects of the custom-built software package, which allows for real-time control of the ROV.

Backend

Waterwitch's backend is built using ROS2 Jazzy. It performs the following jobs:

  • Listening for pilot input and performing necessary thruster math
  • Publishing control commands on the control board's i2c bus or, in the case of the simulation, on Gazebo topics
  • Loading custom pilot control profiles and configuration data from the configuration manager node

Web Frontend

The web frontend application was reused from 2024 and adapted to support waterwitch.

Waterwitch Frontend

The Waterwitch frontend was developed in C++ as an alternative to the web frontend. Although the web frontend makes it possible to control the ROV from any device with a web browser, the Waterwitch frontend was developed for lower latency, quicker development, and more direct control (bypassing ROSlibJS and ROSBridgeServer) that is tailored for Waterwitch.

The graphical part of the application was built using Dear ImGui.

Simulation

The simulation environment was developed as a independent project intended to be used beyond the 2025 competition year. See ROV Simulator for more information.

Streaming Configuration

Waterwitch's live view feed is streamed from the ROV's Raspberry Pi 4 and two aditional Raspberry Pis housed in their own enclosures. The streaming configuration is done with minimal cabling by utilizing ehternet gadget mode for networked communications over USB, where the main enclosure's Raspberry Pi 4 acts similar to an ethernet switch.

Deployment

To ease deployement, the entire application was dockerized and could be installed using a single Docker Compose command.
