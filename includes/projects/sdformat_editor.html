SDFormat Editor

SDFormat Editor is a stand-alone graphical tool for editing SDF (Simulation Description Format) files, notably used by Gazebo robotics simulation. It supports all .sdf files under the SDFormat 14 Specification and provides a 3D viewer for visual and collision elements.

This project was developed alongside by peers, Ean Wheeler and Evan Vokey, for the ECE5100 Software Design Project at the Memorial University of Newfoundland. The course emphasized the use of software design patterns and best practices. See this project's repository

SDF Editor Screenshot


The project is built in C++ using the CMake build system and relies on the following dependencies:

  • libsdformat14 API for SDF parsing, extended by the program's SDFormatParser
  • Dear ImGui for the user interface
  • OGRE for 3D rendering
  • GLM for more complex mathematics operations
  • GoogleTest
