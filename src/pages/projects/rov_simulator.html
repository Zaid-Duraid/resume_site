ROV Simulator

This project demonstrates an implementation of Gazebo Harmonic for simulating underwater remotely operated vehicles (ROVs). It's purpose is to facilitate simulation of underwater ROVs as quickly as possible using open source software. This may help to improve safety, facilitate design validation, and increase testing time. See this project's repository

The simulator is developed such that it can be used by any team for any competition year.

Inspiration

I initally built a previous simulation environment as part of the 2024 Eastern Edge Robotics Software Package. It was built using plugins included in Gazebo Classic 11. I figured that, if only the simulator was available earlier in the competition season, or developed using more tailored plugins and models, it could have been even more useful.
